1. 캐릭터 이름을 변수 값으로 받아 로아 API를 받아와 그 캐릭터의 정보를 보여주는 창을 제작
	a. ArmoryProfile - 캐릭 기본 정보 캐릭사진 , 원정대, 칭호, 서버, 캐릭터 이름 등등
	b. ArmoryEquipment - 캐릭터 착용 장비
		∗ 이름, 타입, 착용부위는 api에 값을 바로 가져 올 수 있지만 그 외 것들을 Tooltip안에서 클롤링으로 찾아야함
			1) tooltip안에 내용을 json으로 받으면 해당 캐릭터가 추가로 가지고 있는 속성에 대한 값만 element 값이 추가되어서 넘어온다.
			2) 해당 값에서 필요한 값을 찾기 위해서 for문과 조건문을 이용하여 사용해야하는 String 값을 찾는다.				ex) [초월] 같은 경우는 1차로 해당 특성이 개방되어 있는 장비를 분류하고 분류되어 있는 장비의 tooltip을 받아 
					 element 값의 name이 “IndentStringGroup” 인 값들을 분류를 한번 더 하고 그 중 에서 value가 있는 값들로 또 한번 분류하게 된다
					 그러면 topStr이란 값 안에 해당 특성을 로아 공식 홈페이지에 띄우기위해 태그값에 넣어 놓은 값이 있다.
					 그 topStr중에서 ‘초월’ 값을 찾기 위해 includes로 확인해서 true일 경우에 
					 그 값을 String으로 받아서 해당 String 값을 search 와 substring, replace를 이용해 필요한 값만 추려낼수 있다.
			3) 장비를 가져와서 이미지 이름 상세 정보를 띄워주기 위해 createElement로 필요 태그를 만들어 하나씩 채우고 
			    해당 장비의 등급에 따라 뒷 배경색이 다른게 적용되는데 filter function을 만들어 뒷 배경색을 자동으로 적용시켜주고
			    해당 장비의 품질에 따라 품질 색상도 위와 같이 자동 분류 작을 적용해준다.
			4) 만약 해당 장비를 미착용 중이라면 비어 있다는 표시를 해주기 위해 악세서리 이미지 부분에 빈 공간만 표시해준 div만 appand해준다
	c. ArmoryEquipment.Emgravings - 캐릭터 장착 각인
		1) 각인 point - true로 활성화 되어 있는 값들을 1,2,3 단계를 구분하여 가져와 표시해준다
	d. ArmoryGem.Gems - 보석
		1) 장착 보석을 배열로 받아와 해당 보석의 배열을 sort로 보석의 순서를 정렬시켜 사용자 입장에서 보기 편하게 정렬 시켜준다
		 	∗ 멸화 홍염 등등 보석 종류로 정렬 후 보석 레밸로 정렬 -> 높은 레밸의 멸화보석부터 내림차순으로 정렬
		2) 해당 보석의 효과 - 능력 스킬 아이콘을 보여주는 ui를 만들어주고 display : none으로 숨겨놓고 
			해당 보석 위에 mouseover를 했을 때 보이고 mouseout를 하면 다시 숨겨지게  이벤트 추가
	e. ArmoryProfile.Stats - 캐릭터 특성
		1) 화면에 띄워주고 싶은 값의 Name 값을 배열로 순서대로 넣어놓고 for문으로 해당 값을 찾아 화면에 띄워준다
			∗ 특성값은 null 값이 없으므로 배열값에 순서대로 넣어 놓으면 항시 같은 순서로 값을 찾아 띄워줄 수 있다.
		2) 성향도 특성과 같이 배열로 값을 찾아오는데 성향값은 max값을 기준으로 해당 값을 백분율로 표시하여 css쪽에 해당 bar의 width값을 변경 하여 보여 줘
			사용자 입장에서 해당 성향이 얼마나 높은지 확인 기능의 가시성을 올렸다.
	f. ArmoryCard - 캐릭터 카드
		1) 장착 카드를 받아와서 이미지, 카드 이름 , 효과를 표시해준다.
		2) 해당 카드의 테두리 각성단계는 카드마다 다르기 때문에 filter 기능이 필요하다
			∗ 데이터의 통신의 데이터량을 감소 시키기 위해 한장의 사진을 div 뒷배경으로 적용시키고 해당 사진에 position을 주고 위치를 변경하는 것으로 테두리를 변경 시킬 수 있다
			∗ 각성 단계도 0단계 5단계가 나와 있는 사진 한장으로 겹쳐 보여주고 awake쪽 사진의 width를 수정하여 해당 카드의 각성 상태를 보여줄 수 있다.
	g. Collectibles - 수집품
		1) 캐릭터 사진 밑에 간단하게 현 수집품 상태를 보여주는 UI를 만들어 주고 해당 UI에 click 이벤트로 상세보기 페이지를 보여주는 이벤트를 만들어준다
			 - 토글기능(클릭시 테두리로 클릭된 상태 표시)
		2) 상세보기 페이지에 수집품 종류별로 퍼센트바로 표시해 주고 해당 수집품바를 선택하면 수집현황을 더 상세히 획득한 수집품 비획득 수집품을 표시해서 보여준다
	h. ArmoryAvatars - 아바타
		1) 캐릭터 사진을 중앙에 배치하고 좌우로 나누어 장착 아바타를 보여주는데 미장착 아바타 슬롯은 미장착으로 표시해 보여준다
		2) 아바타를 순서대로 보여주기 위해 이름과 등급을 묶어 배열에 넣어놓고 그 순서대로 나올 수 있게 처리
		3) 좌 우 아바타 위치를 맞추기 위해 해당 갯수가 append되면 적용되는 순서랑 css가 바뀌게 처리
	i. ArmorySkills - 스킬
		1) 스킬 이미지 표시 오른쪽으로 트라이포드 - 트라이포드 번호, 활성 레벨, 활성 트라이포드 이미지 를 표시해주고 해당 스킬에 장착된 룬을 이미지 등급(색상으로 표시) 이름을 표시
			∗  스킬 보여주는 페이지의 가시성을 위해 해당 스킬의 활성 레벨별로 트라이포드 적용이 다르기 때문에 단계별로 얼마나 많은 빈공간을 만들어줘야하는지 확인해서 
			    항상 룬은 맨 오른쪽에 표시되게 처리
	j. 캐릭터 리스트
		1) 해당 캐릭터 이름으로 해당계정의 캐릭터 리스트를 받아온다
		2) 현제 운영중인 서버 이름을 배열로 만들어 놓고 해당 서버에 있는 캐릭터들을 묶어서 표시할 수 있게 해준다.
		3) 해당 캐릭터를 누르면 그 캐릭터 이름으로 redirect 해준다
